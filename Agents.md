# AGENTS.md

## Setup
- Python 3.12+
- Create venv and install deps:
  - **Windows (PowerShell)**:
    ```
    py -3.12 -m venv .venv
    .\.venv\Scripts\Activate.ps1
    pip install -r backend/requirements.txt
    ```
  - **Linux/macOS/WSL**:
    ```
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r backend/requirements.txt
    ```
